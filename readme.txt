== Description ==

Praktická kalkulačka odchodu do důchodu od Měšec.cz vám během chvíle spočítá důchodový věk. Jednoduchý a rychlý způsob, jak si spočítat, v kolika letech půjdete do důchodu. Zadejte jednoduše do požadovaného pole své pohlaví, datum narození a počet dětí (v případě, že jste žena) a výsledky jsou na světě. Plugin obsahuje odkaz na poskytovatele kalkulačky.

Kalkulačku vložíte do stránky pomocí značky `[kalkulacka-odchodu-do-duchodu]`

== Installation ==
1. V administraci na stránce `Pluginy » Instalace pluginů` vyhledejte kalkulačku a nainstalujte ji, nebo ji stáhněte z webu, rozbalte a nahrajte složku `kalkulacka-odchodu-do-duchodu` do složky `/wp-content/plugins/`.
2. V administraci na stránce `Pluginy » Přehled pluginů` aktivujte plugin "Kalkulačka odchodu do důchodu".
3. Použijte stránku `Nastavení » Kalkulačka odchodu do důchodu` pro nastavení pluginu.
